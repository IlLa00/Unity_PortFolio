# Unity_PortFolio
## 📋 프로젝트 개요
### 프로젝트 소개
**projectlup-v2025**는 6개의 게임 팀으로 나뉘어 협업한 다장르 통합 Unity 프로젝트로 프레임워크 개발자로 참여하여 공통 시스템을 개발했습니다.       
### 프로젝트 정보    
- 개발 엔진: Unity Engine 6000.0.62f         
- 팀 구성: 프로그래머 13명
- 담당 역할: 프레임워크 개발
### 협업 도구
- 버전 관리: Fork
- 문서화: Notion    
## 담당 업무
