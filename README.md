# Unity_PortFolio
## 📋 프로젝트 개요
### 프로젝트 소개
**projectlup-v2025**는 13명의 프로그래머가 6개의 게임 팀으로 나뉘어 협업한 다장르 통합 Unity 프로젝트입니다. 하나의 메타버스 공간에서 서로 다른 장르의 게임들을 자유롭게 전환하며 플레이할 수 있는 독특한 구조를 가지고 있습니다.    
### 팀 구성 및 게임 장르
- **DSG팀** - 덱 전략 게임 (Deck Strategy Game, 다크워서바이벌 스타일)
- **ES팀** - 익스트랙션 슈터 (Extraction Shooter, 리턴얼라이브 스타일)
- **PCR팀** - 생산/건설/자원관리 (Production/Construction/Resource, 다크워서바이벌 스타일)
- **RL팀** - 로그라이크 (Roguelike, 궁수의 전설 스타일)
- **ST팀** - 협동 슈팅 게임 (Shooting, 니케 스타일)
- **FW팀** - 프레임워크 및 공통 시스템 개발
